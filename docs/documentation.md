## Getting started
The framework is installed by maven, you need to choose the modules that you need in

```
<dependency>
  <groupId>com.github.acsgh.mad.scala</groupId>
  <artifactId>core_2.12</artifactId>
  <version>{% current_version %}</version>
  <type>pom</type>
</dependency>
```
